# AssignFlow

## 简介
这是一个用于教师/助教快速记录学生作业提交情况和成绩的桌面工具。
支持学号扫码录入、班级管理、成绩登记、数据导出、历史热力图等功能。
启动后界面简洁，所有操作围绕底部输入框展开。

## 主要功能
- **作业录入**：通过底部输入框扫描学生学号，自动记录提交状态，异常学号可临时添加。
- **成绩录入**：快速为已提交学生登记成绩（支持等级/分数），提供预设按钮。
- **班级学生**：按班级查看所有学生，并支持按时间范围（本周/本月/本学期）查看提交矩阵。
- **导出报告**：将选定班级和时间范围的统计数据导出为 CSV 或 Excel 文件，支持预览和手动调整。
- **AI助手**：提供简单的自然语言指令，如“标记所有未交”、“统计今天情况”等。
- **自动更新**：启动后自动检查 GitHub 最新版本，支持手动触发更新下载。

## 安装与运行
1. 安装 Python 3.8+。
2. 安装依赖：
   ```
   pip install PyQt5 PyQtWebEngine requests openpyxl xlrd
   ```
3. 直接运行本脚本：
   ```
   python 1.py
   ```
4. （可选）打包为独立 exe：
   ```
   pip install pyinstaller
   pyinstaller --onefile --windowed --name "AssignFlow" AssignFlow.py
   ```

## 使用说明
- **首次使用**：先通过“导入花名册”按钮导入包含学号和姓名的 CSV/Excel 文件。
- **新建作业**：每天开始记录前，点击顶部“新建作业”重置今日状态。
- **录入学号**：在底部输入框输入 6 位学号后回车，当前页面会根据所在模式自动处理（作业录入标记提交，成绩录入记录成绩）。
- **切换页面**：点击顶部按钮切换功能模块。
- **查看详情**：在“班级学生”页面点击学生可查看其历史提交热力图和表格。
- **导出数据**：在“导出报告”页面选择班级和时间范围，预览后导出。

## 注意事项
- 学号默认取前4位作为班级（例如 202301 → 2023班），导入时可关闭此功能。
- 数据库文件 `student_data.db` 自动生成在同一目录下，请勿手动修改。
- 软件的最优外设是扫码枪，在每一本作业上贴上对应他们姓名与学号的一维码即可
