# AssignFlow

一个简洁高效的作业管理工具，专为教师设计，帮助记录学生作业提交情况和成绩，支持多班级、多任务，并内置AI助手提供智能问答。

## 功能特性

- **花名册管理**：导入CSV/Excel文件，自动提取学号和姓名，班级可从学号前4位自动生成。
- **作业录入**：扫描学号快速记录提交状态，支持异常学号处理，页面仅显示已输入班级的学生，避免干扰。
- **成绩录入**：为已提交学生记录成绩（分数/等级），支持快速预设。
- **班级学生视图**：按班级查看学生历史提交记录，支持热力图直观展示。
- **数据导出**：导出指定班级和时间范围的作业统计表（CSV/Excel），可手动编辑预览。
- **AI助手**：集成DeepSeek API，可通过自然语言查询学生信息、提交状态、添加学生等（需配置API Key）。
- **自动更新**：从GitHub Release检查并下载更新，支持单文件和多文件模式。
- **数据安全**：SQLite本地存储，支持备份设置，一键清除数据。

## 使用方法

1. **首次使用**：点击“设置”配置DeepSeek API Key（可选），或直接开始录入。
2. **导入花名册**：点击“导入花名册”，选择包含学号和姓名的文件，系统自动映射。
3. **作业录入**：
   - 输入6位学号，系统自动识别班级并显示该班级学生。
   - 已交学生显示在左侧列表，未交学生显示在右侧树形列表。
   - 异常学号（不在花名册）显示在异常列表，可右键添加为新生。
4. **成绩录入**：切换至“成绩录入”页面，设置成绩后输入学号即可记录。
5. **班级学生**：查看各班级学生历史提交情况，支持按时间范围筛选。
6. **导出报告**：选择班级和时间范围，预览后导出。
7. **AI助手**：输入非数字文本自动进入AI对话，可提问如“查询学号202301”“今日统计”等。

## 安装与运行
1. 安装 Python 3.8+。
2. 安装依赖：
   ```
   pip install PyQt5 PyQtWebEngine requests openpyxl xlrd
   ```
3. 直接运行本脚本：
   ```
   python 1.py
   ```
4. （可选）打包为独立 exe：
   ```
   pip install pyinstaller
   pyinstaller --onefile --windowed --name "AssignFlow" AssignFlow.py
   ```

## 注意事项
- 学号默认取前4位作为班级（例如 202301 → 2023班）。
- 数据库文件 `student_data.db` 自动生成在同一目录下，请勿手动修改。
- 软件的最优外设是扫码枪，在每一本作业上贴上对应他们姓名与学号的一维码即可
